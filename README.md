# 专利费用查询助手

一款专为知识产权从业者设计的 Chrome 扩展工具，用于批量查询专利费用信息。从本地 Excel 读取专利号/公告号，通过简单点击填入查询页面，通过简单点击从“费用信息”页面提取费用信息。对于需要留底稿的场景支持打印页面，打印页面时电脑剪切板上自动生成序号+申请号/专利号（如“001-申请号”），不需手动修改打印出来的pdf的文件名。提供5种导出方式，分别是原始json文件、导出最近应缴费用表、导出成本核算表、导出应缴年费表（网上缴费标准格式）、导出全部应缴表。

## 模块介绍

### 1. 批量导入专利号/公告号

- **多格式支持**：Excel（.xlsx、.xls）、CSV 文件
- **智能识别表头**：自动检测表头行位置（不要求在第一行）
- **合并单元格处理**：自动处理 Excel 中的合并单元格
- **智能列名匹配**：自动识别"专利号""申请号""注册号""patent"等列名
- **多 Sheet 支持**：Excel 有多个工作表时可切换选择

### 2. 双模式检索

- **申请号模式**：使用申请号检索专利
- **公告号模式**：使用公告号检索专利
- **一键切换**：在设置页面中切换检索模式
- **智能输入框定位**：三种策略逐层兜底，确保准确填入

### 3. 费用信息提取

- **自动提取**：一键提取页面上的费用信息
- **结构化数据**：分别提取"应缴费用"和"已缴费用"
- **应缴费用字段**：费用种类、应缴金额、缴费截止日、费用状态
- **已缴费用字段**：费用类别、应缴金额、缴费日期、缴费人姓名、票据代码、票据号码
- **累积功能**：支持逐个查询并累积多个专利的费用数据
- **自动跳转**：“提取并累积”完成后自动跳转到下一个专利

### 4. 数据导出

- **JSON 导出**：导出累积的费用数据为 JSON 格式
- **应缴年费表**：生成 Excel 格式的应缴年费汇总表
- **成本核算表**：生成 Excel 格式的成本核算表
- **网上缴费标准格式**：生成符合网上缴费系统标准格式的 Excel 表格
- **全部应缴费用表**：导出所有专利的全部应缴费用记录

### 5. 页面打印

- **原生打印**：使用浏览器原生打印功能，质量高
- **智能命名**：打印前自动将"序号-专利号"复制到剪贴板
- **快捷粘贴**：在保存对话框中 Ctrl+V（Mac 为 Cmd+V）即可粘贴文件名


### 6. 状态管理

- **进度显示**：实时显示处理进度（已处理/总数）
- **专利列表**：显示所有专利号，支持点击快速跳转
- **状态图标**：
  - 🖨️ 已打印
  - 📦 已累积
- **自动恢复**：关闭浏览器后重新打开，状态自动恢复

## 安装方法

### 方式 1：从 GitHub 下载（推荐）

#### 第一步：下载代码

1. 打开 https://github.com/caby-li/patent-fee-query-extension
2. 点击绿色的 **「Code」** 按钮
3. 选择 **「Download ZIP」**
4. 解压到一个**固定文件夹**（建议不要放在临时目录）

#### 第二步：打开 Chrome 扩展管理页面

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车

#### 第三步：开启开发者模式

1. 页面右上角找到「开发者模式」开关
2. 点击开启（开关变为蓝色）

#### 第四步：加载扩展

1. 点击「加载已解压的扩展程序」按钮
2. 在弹出的文件选择器中，选择刚才解压的文件夹
3. 点击「选择文件夹」

#### 第五步：确认安装

1. 扩展列表中会出现「专利费用查询助手」
2. 工具栏中会出现插件图标 
3. 安装完成！

## 使用指南

### 基本工作流程

```
加载 Excel → 选择专利号列 → 填入专利号 → 提取费用信息 → 导出数据
```

### 详细步骤

#### 第一步：加载 Excel 文件

1. 点击工具栏中的插件图标，打开侧边栏
2. 点击「加载专利列表」区域
3. 选择包含专利号的 Excel 文件
4. 等待文件解析完成

#### 第二步：选择专利号列

1. 从下拉菜单中选择包含专利号的列
2. 如果文件有多个 Sheet，可以在下拉菜单中切换
3. 点击「确认选择」

#### 第三步：设置检索模式（可选）

1. 点击标题栏右侧的「设置」按钮 ⚙️
2. 选择检索方式：
   - **申请号**：使用申请号检索（默认）
   - **公告号**：使用公告号检索
3. 点击左上角 ⬅️ 返回主界面

#### 第四步：填入专利号

1. 主界面会显示当前专利号和进度
2. 确保已打开专利查询网站的查询页面
3. 点击「填入专利号」按钮
4. 插件会自动在页面输入框中填入专利号

#### 第五步：提取费用信息

**方式 A：单独提取**
1. 在费用查询页面点击「提取费用信息」
2. 查看提取结果

**方式 B：提取并累积**
1. 点击「提取并累积」按钮
2. 费用信息会自动累积到内存中
3. 进度条会自动跳转到下一个专利
4. 重复此步骤直到处理完所有专利

#### 第六步：打印页面（可选）

1. 点击「打印页面」按钮
2. 浏览器会弹出打印对话框
3. 在保存对话框中，**Ctrl+V**（Mac 为 **Cmd+V**）粘贴文件名
4. 文件名格式为"序号-专利号"，已自动复制到剪贴板
5. 点击保存
6. 插件会自动标记该专利为"已打印"并跳转到下一个
7. 如果并没有成功打印页面，可以在5秒之内通过弹窗撤销标记

#### 第七步：导出数据

1. 点击标题栏右侧的「设置」按钮 ⚙️
2. 选择导出方式：
   - **导出累积 JSON**：导出所有累积的费用数据
   - **导出最近应缴年费表**：生成ddl最近的应缴年费 Excel 表格
   - **导出成本核算表**：生成所有专利的所有已缴费用 Excel 表格
   - **应缴年费表（网上缴费标准格式）**：生成符合向官网批量缴费模板的 Excel 格式的应缴年费汇总表
   - **导出全部应缴表**：生成所有专利的所有应缴费用信息 Excel 表格
3. 文件会自动下载到浏览器的默认下载目录

### 快捷操作

| 操作 | 说明 |
|------|------|
| 点击专利列表项 | 快速跳转到指定专利 |
| 「下一个专利」 | 跳转到下一个专利，不执行其他操作 |
| 「提取并累积」 | 提取费用 + 累积 + 自动跳转下一个 |
| 「打印页面」 | 打印 + 自动标记 + 自动跳转下一个 |
| 🔄 重置按钮 | 重置所有数据，重新开始 |
| ⚙️ 设置按钮 | 进入设置页面 |
| ⬅️ 返回按钮 | 从设置页面返回主界面 |

## 技术说明

### 存储方式

- 使用 Chrome 本地存储（`chrome.storage.local`）
- 数据不会离开您的设备
- 关闭浏览器后数据自动保留

### 权限说明

- `activeTab`：访问当前标签页
- `sidePanel`：显示侧边栏
- `storage`：保存插件状态
- `scripting`：注入内容脚本
- `host_permissions`：在指定网站上运行

### 支持的网站

本插件专为专利查询网站设计，支持以下域名：
- `cpquery.cponline.cnipa.gov.cn`
- `*.cnipa.gov.cn`

### 更新方式

1. 下载新版本的 zip 文件
2. 解压到原文件夹（覆盖旧文件）
3. 在 Chrome 扩展管理页面点击刷新按钮 🔄
