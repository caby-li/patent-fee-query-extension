# 专利费用查询助手

一款专为知识产权从业者设计的 Chrome 扩展工具，用于批量查询专利费用信息。从本地 Excel 读取专利号，自动填入查询页面，支持费用信息提取、累积导出和页面打印。

---

## 功能特性

### 1. 批量导入专利号

- **多格式支持**：Excel（.xlsx、.xls）、CSV 文件
- **智能识别表头**：自动检测表头行位置（不要求在第一行）
- **合并单元格处理**：自动处理 Excel 中的合并单元格
- **智能列名匹配**：自动识别"专利号""申请号""注册号""patent"等列名
- **多 Sheet 支持**：Excel 有多个工作表时可切换选择

### 2. 双模式检索

- **申请号模式**：使用申请号检索专利
- **公告号模式**：使用公告号检索专利
- **一键切换**：在设置页面中切换检索模式
- **智能输入框定位**：三种策略逐层兜底，确保准确填入

### 3. 费用信息提取

- **自动提取**：一键提取页面上的费用信息
- **结构化数据**：分别提取"应缴费用"和"已缴费用"
- **字段完整**：费用种类、金额、缴费截止日、缴费日、收据号等
- **累积功能**：支持逐个查询并累积多个专利的费用数据

### 4. 数据导出

- **JSON 导出**：导出累积的费用数据为 JSON 格式
- **应缴年费表**：生成 Excel 格式的应缴年费汇总表
- **成本核算表**：生成 Excel 格式的成本核算表
- **一键下载**：文件自动下载到浏览器默认目录

### 5. 页面打印

- **原生打印**：使用浏览器原生打印功能，质量高
- **智能命名**：打印前自动将"序号-专利号"复制到剪贴板
- **快捷粘贴**：在保存对话框中 Ctrl+V（Mac 为 Cmd+V）即可粘贴文件名
- **自动跳转**：打印完成后自动跳转到下一个专利

### 6. 状态管理

- **进度显示**：实时显示处理进度（已处理/总数）
- **专利列表**：显示所有专利号，支持点击快速跳转
- **状态图标**：
  - 🖨️ 已打印
  - 📦 已累积
- **自动恢复**：关闭浏览器后重新打开，状态自动恢复

## 安装方法

### 方式 1：加载已解压的扩展程序（推荐）

#### 第一步：下载并解压

1. 下载 `专利费用查询助手-v1.1.0.zip`

1. 扩展列表中会出现「专利费用查询助手」

## 使用指南

### 基本工作流程

```
加载 Excel → 选择专利号列 → 填入专利号 → 提取费用信息 → 导出数据
```

### 详细步骤

#### 第一步：加载 Excel 文件

1. 点击工具栏中的插件图标 📋，打开侧边栏
2. 点击「加载专利列表」区域
3. 选择包含专利号的 Excel 文件
4. 等待文件解析完成

#### 第二步：选择专利号列

1. 从下拉菜单中选择包含专利号的列
2. 如果文件有多个 Sheet，可以在下拉菜单中切换
3. 点击「确认选择」

#### 第三步：设置检索模式（可选）

1. 点击标题栏右侧的「设置」按钮 ⚙️
2. 选择检索方式：
   - **申请号**：使用申请号检索（默认）
   - **公告号**：使用公告号检索
3. 点击左上角 ⬅️ 返回主界面

#### 第四步：填入专利号

1. 主界面会显示当前专利号和进度
2. 确保已打开专利查询网站的查询页面
3. 点击「填入专利号」按钮
4. 插件会自动在页面输入框中填入专利号

#### 第五步：提取费用信息

**方式 A：单独提取**
1. 在费用查询页面点击「提取费用信息」
2. 查看提取结果

**方式 B：提取并累积**
1. 点击「提取并累积」按钮
2. 费用信息会自动累积到内存中
3. 进度条会自动跳转到下一个专利
4. 重复此步骤直到处理完所有专利

#### 第六步：打印页面（可选）

1. 点击「打印页面」按钮
2. 浏览器会弹出打印对话框
3. 在保存对话框中，**Ctrl+V**（Mac 为 **Cmd+V**）粘贴文件名
4. 文件名格式为"序号-专利号"，已自动复制到剪贴板
5. 点击保存
6. 插件会自动标记该专利为"已打印"并跳转到下一个

#### 第七步：导出数据

1. 点击标题栏右侧的「设置」按钮 ⚙️
2. 选择导出方式：
   - **导出累积 JSON**：导出所有累积的费用数据
   - **导出应缴年费表**：生成应缴年费 Excel 表格
   - **导出成本核算表**：生成成本核算 Excel 表格
3. 文件会自动下载到浏览器的默认下载目录

### 快捷操作

| 操作 | 说明 |
|------|------|
| 点击专利列表项 | 快速跳转到指定专利 |
| 「下一个专利」 | 跳转到下一个专利，不执行其他操作 |
| 「提取并累积」 | 提取费用 + 累积 + 自动跳转下一个 |
| 「打印页面」 | 打印 + 自动标记 + 自动跳转下一个 |
| 🔄 重置按钮 | 重置所有数据，重新开始 |
| ⚙️ 设置按钮 | 进入设置页面 |
| ⬅️ 返回按钮 | 从设置页面返回主界面 |

## 使用技巧

### 1. 批量处理流程

```
1. 加载 Excel，选择专利号列
2. 连续点击「提取并累积」处理所有专利
3. 最后在设置中导出数据
```

### 2. 打印命名技巧

打印时，文件名已自动复制到剪贴板：
- 文件名格式：`001-CN202210978848.6`
- 在保存对话框中直接 **Ctrl+V**（Mac 为 **Cmd+V**）粘贴
- 无需手动输入文件名

### 3. 快速跳转

- 点击专利列表中的任意专利号，可快速跳转到该专利
- 适合需要重新处理某个专利的场景

### 4. 数据累积

- 使用「提取并累积」按钮，数据会自动累积
- 累积计数显示在主界面和设置页面
- 导出时会包含所有累积的数据

### 5. 状态恢复

- 关闭浏览器后重新打开，插件状态会自动恢复
- 包括：当前进度、累积数据、打印标记

## 常见问题

### Q: 插件无法填入专利号怎么办？

**A**: 请检查以下几点：
1. 是否已打开正确的专利查询网站
2. 页面是否已完全加载
3. 是否选择了正确的检索模式（申请号/公告号）
4. 尝试刷新页面后重试

### Q: 费用信息提取失败怎么办？

**A**: 请确保：
1. 已打开费用查询页面（不是查询入口页面）
2. 页面上已显示费用信息
3. 如果页面有验证码，需要先手动输入验证码

### Q: 如何切换不同的 Excel 文件？

**A**: 点击主界面右上角的 🔄 重置按钮，然后重新加载新的 Excel 文件。

### Q: 导出的 Excel 文件在哪里？

**A**: 文件会自动下载到浏览器的默认下载目录。可以在 Chrome 设置中查看或修改下载位置。

### Q: 插件会收集我的数据吗？

**A**: 不会。所有数据都存储在您的浏览器本地，不会上传到任何服务器。插件不需要网络连接即可工作。

### Q: 打印时如何快速命名文件？

**A**: 点击「打印页面」后，在保存对话框中直接 **Ctrl+V**（Mac 为 **Cmd+V**）粘贴，文件名已自动复制到剪贴板。

### Q: 如何查看已累积的数据？

**A**: 在主界面底部的「费用信息」区域可以查看当前专利的费用详情。累积总数显示在主界面和设置页面。

### Q: 如何撤销打印标记？

**A**: 打印后会显示一个 toast 通知，点击「撤销」按钮即可取消打印标记。

### Q: 支持哪些 Excel 格式？

**A**: 支持 .xlsx、.xls、.csv 格式。表头可以在任意行，插件会自动检测。

### Q: 如何处理有多个 Sheet 的 Excel？

**A**: 加载文件后，如果检测到多个 Sheet，会弹出选择器让您选择使用哪个 Sheet。

## 技术说明

### 存储方式

- 使用 Chrome 本地存储（`chrome.storage.local`）
- 数据不会离开您的设备
- 关闭浏览器后数据自动保留

### 权限说明

- `activeTab`：访问当前标签页
- `sidePanel`：显示侧边栏
- `storage`：保存插件状态
- `scripting`：注入内容脚本
- `host_permissions`：在指定网站上运行

### 支持的网站

本插件专为专利查询网站设计，支持以下域名：
- `cpquery.cponline.cnipa.gov.cn`
- `*.cnipa.gov.cn`

### 更新方式

1. 下载新版本的 zip 文件
2. 解压到原文件夹（覆盖旧文件）
3. 在 Chrome 扩展管理页面点击刷新按钮 🔄
